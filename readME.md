# Superhero Academy Challenge

Version: 1.0.0-alpha.1

---

## Overview

Superhero Academy Challenge is a web-based multiplayer conditioning game designed specifically for boys gymnastics.

The game is designed to run:

- GitHub Pages
- Embedded inside Google Sites
- Landscape iPad
- Single Device
- Up to 8 simultaneous players

---

## Project Goals

- Fast
- Fun
- Extremely easy for kids
- Coach friendly
- Large readable interface
- Minimal setup
- No scrolling during gameplay
